üîå REST API Endpoints

  • GET /api/v1/orders - Get all ice cream orders
  • POST /api/v1/orders - Create new ice cream order
  • GET /api/v1/orders/{id} - Get specific order by ID
  • GET /api/v1/orders/customer/{email} - Get orders by customer email
  • GET /api/v1/orders/status/{status} - Get orders by status
  • PATCH /api/v1/orders/{id}/status - Update order status
  • POST /api/v1/orders/{id}/cancel - Cancel order
  • GET /api/v1/orders/statistics - Get order statistics
